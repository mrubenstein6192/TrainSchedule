# TrainSchedule

A real-time log of upcoming train times.  The user can add a train based on his/her desired destination, time, and the frequency of the train's arrival.  

## Tools Used

- This app is connected to a Firebase database.

- HTML

- CSS

- Javascript/JQuery

- Moment.js

### Future Improvements
